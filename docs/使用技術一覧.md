# 使用技術一覧

本ドキュメントは、本プロジェクトで実際に利用している技術を目的別に整理したものです。

## 1. 言語・実行環境

- Python 3.11 以上（`pyproject.toml` の `requires-python = ">=3.11"`）

## 2. バックエンド / API

- FastAPI (`fastapi>=0.116.0`)
  - 間違い探し生成 API（`POST /generate`）
  - ヘルスチェック（`GET /health`）
  - 実験トレース参照（`GET /experiments/{trace_id}`）
  - デモ UI 配信（`GET /demo`）
- Uvicorn (`uvicorn>=0.35.0`)
  - ASGI サーバーとして FastAPI を実行
- python-multipart (`python-multipart>=0.0.9`)
  - 画像アップロード（multipart/form-data）処理

## 3. 画像処理・数値計算

- Pillow (`pillow>=10.0.0`)
  - 画像読み込み / 変換 / 保存
  - 差分編集（明るさ・色・コントラスト・シフト・反転）
  - 答え画像生成
- NumPy (`numpy>=1.26.0`)
  - 領域スコア算出
  - 自然さ指標（平均差分・エッジ関連指標）の計算

## 4. セグメンテーション・機械学習

- PyTorch (`torch>=2.4.0`)
  - セグメンテーション推論 / 学習の基盤
- torchvision (`torchvision>=0.19.0`)
  - 画像前処理・モデル周辺処理
- segmentation-models-pytorch (`segmentation-models-pytorch>=0.4.0`)
  - DeepLabV3+ による前景推定
- tqdm (`tqdm>=4.66.0`)
  - 実験スクリプトでの進捗表示

## 5. フロントエンド（デモ）

- HTMX（`/demo` ページで利用）
  - サーバーサイド主導で結果画面を差し替える軽量 UI 構成
- JavaScript / CSS
  - 左右比較表示
  - クリック位置の正解判定とマーキング表示

## 6. 実験・研究運用

- アブレーション実験スクリプト
  - `experiment/ablation_case.py`
  - `experiment/run_ablation.py`
- 自然性分析ベンチマーク
  - `experiment/naturalness_benchmark.py`
- 実験トレース / アーティファクト保存
  - `experiments/YYYYMMDD/trace_id/` 配下に `params.json`, `source.png`, `puzzle.png`, `answer.png` 等を保存

## 7. 開発・実行コマンド

- `uv` ベースの実行（README では `uv run ...` を使用）
  - 例: `uv run uvicorn main:app --host 0.0.0.0 --port 8000`

## 8. 技術選定の意図（要約）

- FastAPI + Uvicorn: API 実装と検証ループを高速に回すため
- Pillow + NumPy: 編集処理と指標計算を軽量に実装するため
- PyTorch + DeepLabV3+: 背景回避と前景重視の領域選択を可能にするため
- HTMX: 研究デモを少ない実装コストで提供するため
